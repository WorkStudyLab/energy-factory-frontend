#### 1.4 단위당 가격 계산 시스템 추가
```typescript
// 스마트 단위 결정 로직
const getUnitPrice = (price: number, weight: number) => {
  let unit = weight >= 100 ? 100 : 10;
  let label = weight >= 100 ? '100g당' : '10g당';
  const unitPrice = Math.round((price / weight) * unit);
  return { price: unitPrice, label };
};
```

**표시 예시**:
```
45,000원 (500g)
500g • 100g당 9,000원
```
